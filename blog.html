Question-1: What are the different ways to select an element in the DOM?

DOM এ একটি Element নির্বাচন করার জন্য- By ID, By Class Name, By Tag Name, By Query Selector, By QuerySelectorAll এগুলো ব্যবহার করা হয়।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML দিয়ে সাধারনত একটি Element এর ভিতরে HTML সেট করে এবং কোনো HTML কে পেতে গেলে innerHTML লাগে। অন্যদিকে innerText Element এর ভিতরের মানকে রিটার্ন করে অথবা সেটও করে। যখন কোনো মানের প্রয়োজন তখন innerText ব্যবহার করা হয়। textContent এইটা সম্পর্কে ভালো জানি না।

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন হলো জাভাস্ক্রিপ্টের একটি কৌশল যেখানে একটি ইভেন্ট লিসেনারকে চাইল্ড এলিমেন্টে একাধিক লিসেনার যোগ করার পরিবর্তে একটি প্যারেন্ট এলিমেন্টের সাথে যুক্ত করা হয়। যেখানে ইভেন্টগুলি চাইল্ড থেকে প্যারেন্ট এলিমেন্ট হয়।

Question-4: What is event bubbling in the DOM?

ইভেন্ট বাবলিং হল ইভেন্ট প্রচারের একটি পর্যায় যেখানে একটি ইভেন্ট টার্গেট এলিমেন্ট থেকে শুরু হয় এবং DOM এর মূল এলিমেন্টগুলির উপরে চলে যায়।

Question-5: How do you create, add, and remove elements using JavaScript?

জাভাস্ক্রিপ্টে DOM API ব্যবহার করে Element create , add এবং remove করা যায়।
